Este é um teste de Autoavaliação das Características do Comportamento Empreendedor. O objetivo é identificar os comportamentos empreendedores que você possui e que podem ser desenvolvidos.

Autoavaliação

Para preencher a autoavaliação reflita e escolha o número que melhor descreva a sua prática no dia a dia, conforme cada comportamento listado.

Critérios

  1. Nunca pratico este comportamento
  2. Raramente pratico este comportamento
  3. Às vezes pratico este comportamento
  4. Muitas vezes pratico este comportamento
  5. Sempre pratico este comportamento
